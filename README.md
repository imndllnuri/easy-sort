# Easy Sort

A lightweight, vanilla-feel inventory and chest sorting mod for Minecraft.

One hotkey sorts a container using a stable, configurable ordering and merges
partial stacks — no inventory-management suite bloat, no forced GUI overhaul.

## Status

Early development (pre-MVP). See [ROADMAP.md](ROADMAP.md) for planned features
and milestones.

## Supported versions

| Minecraft | Loader | Status |
|---|---|---|
| 1.21.10 | Fabric | In development |

NeoForge/Forge support is planned for a later release (see [ROADMAP.md](ROADMAP.md)).

## Installation

Not yet published. Once released, Easy Sort will be available on
[Modrinth](https://modrinth.com) and [CurseForge](https://www.curseforge.com).

Requires [Fabric Loader](https://fabricmc.net/) and
[Fabric API](https://modrinth.com/mod/fabric-api).
